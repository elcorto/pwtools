pwtools.rbf.hyperopt.FitError.err_direct¶

FitError.err_direct(params)[source]¶

Normal fit error w/o CV. Uses Rbf.fit_error().

Build and Rbf model with self.rbf_kwds, fit, return the fit error (scalar, sum of squares). Should be zero for interpolation, i.e. no regularization r=0.
