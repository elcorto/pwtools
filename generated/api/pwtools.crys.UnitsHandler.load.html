pwtools.crys.UnitsHandler.load¶

UnitsHandler.load(dump_filename)¶

Load pickled object.

Examples

>>> # save
>>> x = FileParser('foo.txt')
>>> x.parse()
>>> x.dump('foo.pk')
>>> # load: method 1 - recommended
>>> xx = io.read_pickle('foo.pk')
>>> # or
>>> xx = pickle.load(open('foo.pk'))
>>> # load: method 2, not used / tested much
>>> xx = FileParser()
>>> xx.load('foo.pk')
