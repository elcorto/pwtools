pwtools.num.vlinspace¶

pwtools.num.vlinspace(a, b, num, endpoint=True)[source]¶

Like numpy.linspace, but for 1d arrays. Generate uniformly spaced points (vectors) along the distance vector connecting a and b.

Parameters

  • a (1d arrays) –

  • b (1d arrays) –

  • num (int) –

Return type

array (num, len(a)), each row is a “point” between a and b
