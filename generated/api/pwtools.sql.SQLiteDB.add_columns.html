pwtools.sql.SQLiteDB.add_columns¶

SQLiteDB.add_columns(header, table=None)[source]¶

Convenience function to add multiple columns from header. See get_header().

Parameters:

  • header (sequence) – see get_header()

  • table (str, optional)

Examples

>>> db.add_columns([('a', 'text'), ('b', 'real')])
# is the same as
>>> db.add_column('a', 'text')
>>> db.add_column('b', 'real')
